=== Ancien Code de Seeding : init_db.py ===
Usage : Création automatique du compte administrateur/agent de test par défaut.
```python
def seed_default_agent():
    db = SessionLocal()
    try:
        email = "agent1@gmail.com"
        existing_user = db.query(User).filter(User.email == email).first()
        if not existing_user:
            default_agent = User(
                nom="Amani",
                prenom="Manasse",
                email=email,
                telephone="0714781378",
                password_hash=hash_password("agent@23"),
                role=UserRole.AGENT_ADMINISTRATIF,
                is_active=True
            )
            db.add(default_agent)
            db.commit()
    except Exception as e:
        db.rollback()
    finally:
        db.close()
```
